NodeMCU module

Description

The development of a Smart Home Automation System, a comprehensive project aimed at enhancing convenience and control over household appliances through the integration of modern hardware and software technologies. The system utilizes NodeMCU, Sinric Pro, and Google Home to provide a seamless user experience.

The primary objective of this project is to create a smart home automation system that allows users to control and monitor their household appliances using hand gestures, voice commands, a web application, and a mobile application. The system is designed to be user-friendly, efficient, and reliable, ensuring maximum convenience and ease of use for the end-users.

The system captures and interprets hand gestures using Python-based recognition technology. Users can perform specific gestures to control various appliances, providing a touch-free method of interaction that enhances convenience and accessibility.Voice commands are processed using Python to enable control over home appliances. This feature allows users to operate their devices simply by speaking, which is particularly useful in situations where manual control is inconvenient or impossible.The web application offers a user-friendly interface that allows users to manage and schedule the operation of their home appliances from any device with internet access. It displays the current status of each appliance and provides easy-to-use controls for turning devices on or off.The mobile application provides an intuitive interface optimized for touch interaction on iOS and Android devices. It supports real-time updates on appliance status and allows users to control their appliances on the go.
